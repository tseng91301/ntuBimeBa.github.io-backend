## 功能需求整理 (API行為) ##

| 功能             | API 路徑                          | 方法     | 說明              |
| -------------- | ------------------------------- | ------ | --------------- |
| 取得所有資料夾列表      | `/api/legacy/folders`           | GET    | 回傳所有資料夾名稱與 ID   |
| 取得指定資料夾的檔案     | `/api/legacy/folder/:id/files`  | GET    | 回傳該資料夾內所有檔案資訊   |
| 新增資料夾          | `/api/legacy/folder`            | POST   | 建立新資料夾（可選擇限制角色） |
| 上傳檔案至資料夾       | `/api/legacy/folder/:id/upload` | POST   | 上傳檔案至該資料夾       |
| 刪除檔案或資料夾（選擇實作） | `...`                           | DELETE | 可選做             |
